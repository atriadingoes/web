🔍 Диагностика сети и блокировок

Проверка доступности доменов и анализ блокировок оператора

📱 Информация о клиенте

Собираем информацию о вашем подключении: IP адрес, тип сети (4G/5G), скорость и задержку. Это поможет понять характеристики вашего мобильного интернета.

🌐 Диагностика блокировок оператора

Проверяем какие протоколы блокирует ваш оператор. HTTP/2 обязателен для Reality протокола. Блокировка WebSocket или UDP может влиять на работу других VPN протоколов (Shadowsocks, WireGuard).

📋 Проверка доступности SNI доменов

Проверяем какие домены из whitelist доступны с вашего мобильного интернета. Доступные домены можно использовать как SNI для Reality протокола. Проверка идет параллельно батчами по 30 доменов для ускорения.
